[ 07 ] Questions

Does this work with my Claude Max plan?

Yes — subscription traffic passes through untouched and stays free, and it's still fully logged. Routing to a different model uses a metered key instead, so those calls are billed by whichever provider serves them.

How do you know which model really answered?

It's parsed from the provider's own response, never copied from the request. You can cross-check any call against your provider's billing dashboard — the token counts line up exactly.

Can I run it with no internet at all?

Yes. Point it at a local model and turn on offline mode — every timer-driven outbound call stands down. Verified by sampling the process's sockets across a full cycle: nothing but localhost.

Is a small local model actually usable?

For self-contained work, often yes — a local 4-bit MoE beat a frontier model on a single-file build here. Long multi-file agent loops are where frontier models still pull ahead. Pick a model whose KV cache holds, or you pay the harness cost on every turn.

What does AGPL mean for me?

Self-host it freely, forever. If you run a modified version as a service for others, publish your changes. A commercial licence is available if you'd rather not.
