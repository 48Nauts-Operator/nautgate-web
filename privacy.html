Privacy

Last updated 22 July 2026

This website

nautgate.dev is a static site served from GitHub Pages. It sets no cookies and asks for nothing.

We use Umami, self-hosted on our own infrastructure, to count page views. It is cookieless and stores no personal data — no cross-site tracking, no advertising profiles, no data sold or shared. Aggregate counts only: which pages were viewed, roughly where from, and which site referred you. Block it with any content blocker and the site works identically.

GitHub serves the files and, like any web host, sees your IP address in its request logs. That is covered by GitHub's privacy statement. The version number in the header is fetched from GitHub's public API by your browser, which means GitHub sees that request too.

The software

NautGate itself is something you run. We do not host it, cannot see it, and receive no telemetry from it. There is no phone-home, no licence check, no usage beacon — offline mode exists precisely so that this is verifiable at the socket rather than merely asserted here.

Whatever NautGate records — prompts, responses, costs, routing decisions — is stored in your own database on your own machine, governed by the capture policy you configure. None of it reaches us.
